#  Sales & Demand Forecasting using Time-Series Analysis

##  Project Overview

This project focuses on forecasting future sales using historical store sales data. Time-series analysis techniques are applied to identify trends and predict future demand, helping businesses make data-driven decisions.


##  Objective

To build a forecasting model that predicts future sales based on past sales data using time-series modeling techniques.

##  Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Statsmodels (ARIMA Model)


##  Methodology

1. Data Loading and Exploration
2. Data Preprocessing (Date Conversion, Cleaning)
3. Aggregation of Daily Sales
4. Time-Series Visualization
5. Model Building using ARIMA
6. Forecasting Future Sales
7. Model Evaluation using RMSE and R² Score


##  Results

* The model was evaluated using RMSE (Root Mean Squared Error) and R² Score.
* Results indicate moderate predictive performance due to the complexity and variability of real-world sales data.

##  Dataset

Dataset used: **Store Sales Dataset (Kaggle)**

>  Note: The dataset is not included in this repository due to size limitations.
> Please download it from Kaggle and place it in the same directory before running the notebook.

##  How to Run

1. Download the dataset from Kaggle
2. Place `train.csv` in the project folder
3. Open the notebook (`task1_sales_forecasting.ipynb`)
4. Run all cells sequentially
