# Sales Forecasting using Regression:

# Overview
This project focuses on predicting sales using historical data. It involves data cleaning, time-based feature engineering, regression modeling, and performance evaluation.
The goal is to generate insights and forecasts that can be easily understood by non-technical stakeholders.

# Steps:
- Data cleaning and preprocessing
- Time-based feature engineering (month, day, etc.)
- Model building using Linear Regression
- Model evaluation using RMSE and R²
- Visualization of actual vs predicted sales
- Error analysis and trend visualization

# Tools & Technologies:
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

# Results:
- The model captures overall sales trends.
- RMSE is used to measure prediction error.
- R² score indicates how well the model explains variability.
- Visualizations help in understanding predictions clearly.

# Visualizations:
- Actual vs Predicted Sales
- Error Distribution
- Monthly Sales Trend (Seasonality)
